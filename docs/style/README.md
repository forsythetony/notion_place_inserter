# Style (product visual direction)

This folder holds **product-level** visual direction and pre-token exploration for the app. It is not the source of truth for implementation details.

## Canonical style guide (implement here)

Tokens, components, layout, pipeline editor behavior, and guardrails live in the **Notion Pipeliner UI** repository:

| Where | What |
|-------|------|
| Local clone (typical layout) | Open `notion_pipeliner_ui/styleguide/README.md` next to this repo (sibling directory under the same parent). |
| Entry point | Start with `styleguide/README.md`, then the six topic files (`design-principles.md`, `layout-and-navigation.md`, etc.). |

Treat that styleguide as the contract for UI work; this backend repo only keeps historical direction and cross-links.
